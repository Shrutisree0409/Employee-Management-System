# Employee-Management-System
A console-based Employee Management System built using Python with features to add, view, and search employee records.

### Overview

This is a console-based Employee Management System developed using Python. The application allows users to manage employee records through a menu-driven interface.

### Features
Add Employee
View All Employees
Search Employee by ID
Exit Application

Dictionaries
Functions
